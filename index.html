ABOUT MAPS MARKET TOOL

Welcome to the Moving Average Points System (MAPS) Market Tool.

MAPS is derived from datapoints that a separate tool (the Moving Average Cluster Analysis (MACA) Market Tool) calculates from candlestick price data.

MACA processes candlestick data into a series of datapoints using a user-defined range of Simple Moving Averages (SMAs). Once calculated, MACA uses SMA price data (backdrop) to discover SMA event data (MACA-1). MACA-1 is accurate in milliseconds to 1/1000 of a penny...

According to a user-specified time interval, MAPS histograms the backdrop and MACA-1 separately to determine their point-values. Percentage differences between the point-values of the two histograms determine the MAPS histogram values. These values oscillate positive when MACA-1 histogram is greater than the backdrop histogram and negative when the opposite. The result is a histogram with unbound percentage data that oscilates positive and negative.

Implementations:

Carry out normal technical analysis (TA).

All code is written in Python and uses readily accessible market data. The code can process any asset at any time interval.

MACA-1 data is created and saved in binary (.pickle) using NumericalPython. Data is easily accessible and machine learning algorithms can process the data.

MAPS and MACA-1 histogram data are saved in a similar binary format.

Limitations:

Currently the processing and graphing functions lack any real-time refresh functions. Data is processed in bulk and update functions must reprocess the entire last set of bulk data.

There has been no outside review for accuracy. Minor increments of error might still exist evenly throughout the histograms.
